# LeetCode SQL Problems

Problems are organized by **difficulty** and **topic**.

## Structure

```text
leetcode_problems/
├── easy/
│   ├── joins/
│   └── date_functions/
├── medium/
│   ├── aggregation/
│   └── sequences/
└── hard/
    ├── sequences/
    └── window_functions/
```

## Naming

- SQL: `<id>_<slug>.sql`
- Optional notes: `<id>_<slug>.md`
